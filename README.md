# milan-fish-plugin

A personal Fish shell plugin for Milan.

## Abbreviations

- `l` — `eza -l` (long listing)
- `ll` — `eza -lah` (long listing, all files, human-readable sizes)
- `glm` — `git pull --rebase origin <main-branch>`

## Requirements

- [eza](https://github.com/eza-community/eza) must be installed (`brew install eza`)
- [jhillyerd/plugin-git](https://github.com/jhillyerd/plugin-git) (provides `__git.default_branch`)

## Installation

```sh
fisher install sukovanej/milan-fish-plugin
```
